# Conclusiones

Este proyecto ha permitido la aplicación práctica de varias herramientas y conceptos fundamentales 
en el desarrollo de software y la gestión de la infraestructura.

1. Gestión de Proyectos con Git y GitHub

2. Documentación y Comunicación con Markdown

3. Despliegue y Contenerización con Docker y NGINX

4. Automatización con GitHub Actions

En resumen, la combinación de estas herramientas forma un ecosistema eficiente para el desarrollo moderno, 
desde la codificación y control de versiones hasta el despliegue automatizado y la entrega de servicios.
